# Utils

## qBittorrent Mover

Follow instructions from [TRaSH guides](https://trash-guides.info/Downloaders/qBittorrent/Tips/How-to-run-the-unRaid-mover-for-qBittorrent/).
Cited scripts are saved locally.

Summary:

1. Install user scripts plugin if not already installed
2. Add `install-qbittorrent-api.sh` script and run in background
3. Upload the `mover.py` script to an appropriate location in app data
4. Add the `orchestrator.sh` script and run it on a schedule
